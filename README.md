## About Project
### **소프티어 부트캠프 3기 - 워밍업 프로젝트 소프티5, 15지조 팀**
> **개발 기간: 2024.01.04 ~ 2024.01.10**

#### 제주도 여행객들의 친목도모를 위한 차량 공유 모빌리티 서비스

</br>

## 기능
### 1. 간단한 일회성 회원 등록 📝
닉네임, 성별, 나이, 사진, 자동차 소유 여부 입력해서 회원 등록하기
### 2. 지도에서 상대방 선택 🗺️
뚜벅이는 운전자만, 운전자는 뚜벅이만 보이는 지도에서 상대방 선택하기
### 3. 카팅 매칭 ♥️
마음에 드는 사람과 카팅 매칭

</br>
